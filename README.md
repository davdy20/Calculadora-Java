# 🧮 Calculadora Simples em Java

Este projeto é uma calculadora simples desenvolvida em Java com o objetivo de praticar lógica de programação e consolidar conceitos fundamentais da linguagem.

> 💡 Este projeto foi desenvolvido com o apoio de inteligência artificial como ferramenta de aprendizado.

## 🚀 Funcionalidades

- Soma
- Subtração
- Multiplicação
- Divisão

## 🛠️ Tecnologias utilizadas

- Java
- Scanner (entrada de dados pelo usuário)

## 💡 Objetivo do projeto

Este projeto foi desenvolvido com foco em:

- Praticar lógica de programação
- Trabalhar com estruturas condicionais
- Utilizar entrada e saída de dados
- Melhorar a organização do código
